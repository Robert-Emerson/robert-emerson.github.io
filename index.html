About Me

I'm a graduate of the University of Virginia, with a Bachelors of Science in Computer Science from the School of Engineering and Applied Sciences. My interests include highly parallel distributed systems, databases, and file systems. I like making pretty things, and things that go fast intelligently.

In addition to my interest in computing, I can often be found woodworking, hiking, photographing, baking or enjoying the comfort of a couch while losing a race in Mario Kart.

Projects

Many designed and built for classes throughout my college years.

Dropbox Clone

Dropbox-like Unix application that allows file syncing between multiple computers and a server. Built for UVa CS 3240 - Advanced Software Development using Python and the Flask framework.

Twitter Data Analysis

Map-Reduce Python program designed to gauge Twitter user sentiment on the almost ousting of UVa President Teresa Sullivan in June 2012 by scanning 50,000+ archived tweets. Built for UVa CS 4501 - Cloud Computing.

Sites To See

Travel web app that allows users to search for interesting places to visit in a given location, then share itinerary via text. Built for hack.uva 2014 and winner of the Twilio API prize.

And More!

There are more projects on my GitHub page! I've worked on a wide range of projects, from a C++ implementation of the Karger Klein Tarjan MST algorithm to a photo scavenger hunt Android application.
